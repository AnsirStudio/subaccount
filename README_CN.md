# JioJio — 订阅与账号管理

管理你的订阅，追踪你的账号，一切都在本地。

大多数 App 只帮你记录订阅。**JioJio** 不止于此 —— 它还能帮你管理每个订阅背后的**账号和登录方式**。你用哪个邮箱注册的？用的微信还是 Google 登录？同一个平台有几个账号？JioJio 全部帮你记清楚，再也不用到处翻找。

## 为什么需要 JioJio？

**你的订阅比想象中多。** 视频、音乐、AI 工具、云存储、开发服务……加起来一大堆。每个都有不同的计费周期、不同的货币、不同的支付方式。

**你记不清用了哪个账号登录。** 有的用 Google，有的用微信，有的用工作邮箱。同一个平台甚至有多个账号。JioJio 帮你把每个账号和每个订阅一一对应，再也不搞混。

**你的数据只属于你。** 所有数据存储在本地设备上，无需注册账号，无需云端同步，不会收集任何数据。

## 功能特性

### 订阅管理
- **50+ 内置服务模板**，涵盖视频、AI、开发、云服务、工具、音乐、社交、购物 9 大类别，持续更新中
- **自定义订阅** —— 手动添加任意服务，不受模板限制
- **灵活计费周期** —— 每月、每年、自定义天数
- **自动续费追踪** 与 **到期提醒**（当天、提前 1/3/7 天）
- **置顶重要订阅**，一目了然

### 账号与登录管理
- 记录每个订阅的**登录方式** —— 手机、微信、邮箱、QQ、Gmail、Apple ID、GitHub
- 存储**账号标识** —— 精确知道哪个账号绑定了哪个服务
- 管理**同一平台的多个账号**，不再混淆

### 财务总览
- **仪表盘** —— 当月费用、年化成本、即将到期
- **多币种支持**（10+ 常用货币），自动折算为用户指定的主币种
- **类别占比分析** —— 看清钱花在哪里
- **现金流时间线** —— 过去 12 个月实际支出 + 未来 3 个月预测

### 视图与筛选
- **卡片视图** 与 **表格视图**
- **排序** —— 按结束日期、开始日期、月费、年费
- **筛选** —— 计费周期、支付方式、类别、续费状态、提醒状态
- **搜索** —— 快速查找订阅

### 个性化设置
- **双语界面** —— 中文 / English
- **主题切换** —— 跟随系统 / 浅色 / 深色
- **自定义汇率** —— 手动调整币种转换比率

## 快速开始

### 环境要求

- [Node.js](https://nodejs.org/) (v18+)
- [Rust](https://www.rust-lang.org/tools/install)（最新稳定版）
- [Tauri 依赖](https://v2.tauri.app/start/prerequisites/)

### 安装与运行

```bash
git clone https://github.com/AnsirStudio/JioJio.git
cd JioJio
npm install
npm run tauri dev
```

### 构建打包

```bash
npm run tauri build
```

## 数据存储

所有数据存储在浏览器的 **localStorage** 中。不会向任何服务器发送数据，无需注册账号。
